100

表示限定大矩形的宽度为100，输入6个小矩形，它们的宽和高分别是95/5, 80/10, 51/10, 23/15, 20/17, 65/10。
输入的小矩形的宽和高可以交换。输入结束后打开白色的页面在里面单击左键，可以看到小矩形是怎么布置的（大矩形是整个窗口）。demo似乎在64位系统中才能打开。

用于测试时间复杂度的源代码在texturePackingForTest文件夹，只需要输入大矩形的宽度和小矩形的总数就可以运行，我测的总时间复杂度是O(N^2)。
